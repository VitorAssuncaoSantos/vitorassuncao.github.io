Atividades para praticar algoritmos

Faça um programa que receba a idade e sexo de um número indefinido de pessoas. Ao fim, seu programa deve informar

  • O número de maiores de idade
  • O número de melhores
  • O número de homens
  • Média de idade total
  • Média de idade de homens
  • Média de idade de mulheres
